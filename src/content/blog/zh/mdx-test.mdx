## 为什么选择 MDX？

MDX 是一种特殊的 Markdown 变体，它支持嵌入 JavaScript 和 JSX 语法。这开启了在 [Markdown 内容中组合 JavaScript 和 UI 组件](https://docs.astro.build/en/guides/markdown-content/#mdx-features) 的能力，适用于创建交互式图表或警告框等。

如果你已有使用 MDX 编写的内容，该集成将使你能够轻松地迁移到 Astro。

> [!IMPORTANT]
> 我创建这个插件是为了个人使用 —— 目前还没有时间对其进行深度打磨。我建议你暂时谨慎使用，如果使用了，请留意随时可能发生的破坏性更改或重构。
