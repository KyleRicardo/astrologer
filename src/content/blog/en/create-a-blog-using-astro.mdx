## Hexo Migration Guide

1. **Content Migration**  
   Copy the Markdown files from the `source/_posts` directory to `src/content/posts`.

2. **Template Transformation**  
   Convert `.ejs`/`.swig` templates into `.astro` components:
   ```astro
   <!-- Original Hexo template snippet -->
   <%- partial('_partial/article', { post: page.posts }) %>

   <!-- Converted Astro template -->
   <Article {post} />
   ```

3. **Generator Configuration**  
   Use the `@astrojs/rss` package to replace `hexo-generator-feed`.
